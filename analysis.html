Web Design Analysis

1. Appealing Website Example

a. URL of the website: https://www.apple.com

b. Does this page use tables? No, it does not use tables for layout. It uses modern semantic elements and CSS layouts.

c. Does this page use CSS? Yes, Apple uses high-end CSS for typography, responsive design, and smooth layout configurations across all devices.

d. Is this page appealing or unappealing? This page is highly appealing. Three reasons for this are:

  • Clean Minimalism: Use of high-contrast white space makes the products stand out.
  • Professional Typography: The fonts are clear, consistent, and easy to read.
  • Visual Hierarchy: Important information and beautiful product images are organized perfectly to guide the eye.

e. Improvement: N/A (The page follows industry-leading design standards).

2. Unappealing Website Example

a. URL of the website: http://www.pnwx.com/

b. Does this page use tables? Yes, the entire structural layout is built using old HTML tables, which is obsolete.

c. Does this page use CSS? No, it relies on outdated inline HTML attributes for styling rather than external CSS files.

d. Is this page appealing or unappealing? This page is highly unappealing. Three reasons for this are:

  • Poor Color Scheme: The bright blue and green text on dark backgrounds is hard to read.
  • Cluttered Layout: Too many links and images are packed together without proper spacing.
  • Outdated Graphics: The low-resolution icons and old-style buttons make the site look unprofessional.

e. What would you do to improve it? I would remove the table layout and use a modern CSS Grid. I'd also simplify the color palette to a professional white background with dark text and organize the many links into a clean navigation menu.
