Web Design Analysis

1. Appealing Website Example

a. URL of the website: https://www.firewatchgame.com/

b. Does this page use tables? No, it does not use tables. The layout relies on modern semantic HTML tags (like sections and divs) rather than outdated table structures.

c. Does this page use CSS? Yes, it makes excellent use of CSS. It uses CSS for a beautiful "parallax scrolling" effect, custom typography, spacing, and making the site fully responsive for mobile devices.

d. Is this page appealing or unappealing? This page is highly appealing. Three reasons for this are:

  • Immersive Depth: The CSS parallax scrolling creates a 3D illusion that draws the user into the webpage.
  • Excellent Color Palette: The warm, sunset-themed colors are visually striking and perfectly reflect the product's mood.
  • Minimalist Interface: The content is not crowded; call-to-action buttons are clear and easy to find without distracting text.

e. Improvement: N/A (The page functions perfectly as a modern, artistic landing page).

2. Unappealing Website Example

a. URL of the website: http://www.pnwx.com/ (Pacific Northwest X-Ray Inc.)

b. Does this page use tables? Yes, the entire structural layout of the page is built using rigid HTML tables. This is an obsolete practice that breaks the layout on smaller screens.

c. Does this page use CSS? No, it barely uses any external CSS. The design relies heavily on outdated inline HTML attributes (like bgcolor and font tags) which makes the code repetitive and messy.

d. Is this page appealing or unappealing? This page is highly unappealing. Three reasons for this are:

  • Terrible Color Contrast: The neon green and harsh blue text on dark backgrounds cause immediate eye strain.
  • Information Overload: The homepage is a massive, unorganized wall of text and links with no clear hierarchy.
  • Lack of Responsiveness: Because of the fixed table layout, users on mobile phones have to scroll horizontally to read anything, ruining the user experience.

e. What would you do to improve it? To improve this site, I would completely remove the table-based layout and replace it with a modern CSS Flexbox or Grid system. Secondly, I would change the color scheme to a clean, professional white background with dark gray text to ensure readability. Finally, instead of putting all links on the homepage, I would organize them into a clean dropdown navigation menu at the top.
