Web Design Analysis

1. Appealing Website Example

a. URL of the website: https://www.apple.com

b. Does this page use tables? No, it utilizes modern semantic HTML and CSS layouts instead of tables.

c. Does this page use CSS? Yes, it uses advanced CSS for responsive grid layouts, typography, and visual consistency.

d. Is this page appealing? Yes. Three reasons: Clean use of white space, high-resolution product imagery, and an intuitive navigation system.

2. Unappealing Website Example

a. URL of the website: http://www.pnwx.com/

b. Does this page use tables? Yes, the entire structural layout is built using outdated HTML tables.

c. Does this page use CSS? No, it relies heavily on deprecated inline HTML attributes and font tags for styling.

d. Is this page unappealing? Yes. Three reasons: Harsh color contrast (blue/green on dark), cluttered content delivery, and a total lack of mobile responsiveness.

e. Improvement: I would re-structure the site using a modern CSS Flexbox layout and implement a professional, high-contrast color palette to improve readability.
