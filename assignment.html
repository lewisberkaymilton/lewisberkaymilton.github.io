HW3 Q1 and Q2 HTML & CSS Code Fixes

1. Missing Code Completion

Task: Fill in the missing code so that the background and text colors have good contrast, and the header area uses Arial.

Solution: Since the background color is a dark blue (#0066CC), the text color is set to white (#FFFFFF) for optimal contrast. The header font is set to Arial. Also, the missing closing tags for </style> and </head> are added.

<!DOCTYPE html>
<html lang="en">
<head>
<title>Trillium Media Design</title>
<meta charset="utf-8">
<style>
body { 
    background-color: #0066CC;
    color: #FFFFFF; /* High contrast text color */
}
header { 
    font-family: Arial, sans-serif; /* Arial font for header */
}
</style>
</head>
<body>
<header><h1>Trillium Media Design</h1></header>
<p>Our professional staff takes pride in its working relationship with our clients by offering personalized services
that listen to their needs, develop their target areas, and incorporate these items into a website that works.</p>
</body>
</html>

2. Find the Error

Task: Why won’t the page corresponding to the provided code display properly in a browser?

Solution: The error is located at the end of the CSS block. The <style> tag is mistakenly opened again instead of being closed. It should be written as </style> with a forward slash. Without the slash, the browser cannot determine where the CSS ends and the HTML begins, which breaks the page layout.

<!DOCTYPE html>
<html lang="en">
<head>
<title>Trillium Media Design</title>
<meta charset="utf-8">
<style>
body { 
    background-color: #000066;
    color: #CCCCCC;
    font-family: Arial,sans-serif;
    font-size: 1.2em; 
}
</style> <!-- The error was here, forward slash added -->
</head>
<body>
<header><h1>Trillium Media Design</h1></header>
<main>
    <p>Our professional staff takes pride in its working relationship with our clients by offering personalized
    services that listen to their needs, develop their target areas, and incorporate these items into a website that
    works.</p>
</main>
</body>
</html>
